## üí® Deliverable 1: Air Quality Prediction Models

### Overview
This deliverable applies machine learning techniques to real-world environmental data to predict and analyze air quality levels across global regions.

### Dataset
- **World Air Quality Index by City and Coordinates** (Kaggle)
- Contains numeric pollutant measurements used to calculate AQI values

### Methods & Concepts
- Supervised learning models for AQI prediction
- Unsupervised clustering to identify regional air-quality patterns
- Model performance comparison

### Installation Instructions
1. Navigate to the `Deliverable 1: Air Quality Prediction Models/` folder.
2. Ensure Python is installed (Python 3.x).
3. Install required dependencies listed in the notebook

### Usage Instructions
- Open the Jupyter Notebook in the folder.
- Run all cells in order to view data preprocessing, model training, and evaluation.
- Results and comparisons are explained directly in the notebook.
